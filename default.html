현재 지원하는 URL 스킴 목록:

  • HTTP: http://example.com
  • HTTPS: https://example.com
  • File: file:///path/to/local/file.html
  • Data: data:text/html,<h1>Hello</h1>

사용 방법

python3 url.py http://example.org
python3 url.py file:///Users/username/Documents/test.html
python3 url.py "data:text/html,hello world!"
